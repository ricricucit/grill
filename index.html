This project born by chance, while I was dusting my rusty coding skills, aiming to design more in browser and less in illustrator.
I tried some famous and unknown grid frameworks available on internet, it was horrible, i felt like i was designing a book with excel instead indesign.
So i started to write this tiny sass library to use for my responsive prototypes.
Here we are, it is not finished nor distribution ready, but it's here so that you can help by contributing.
